About jQuery

jQuery — библиотека JavaScript,

фокусирующаяся на

взаимодействии JavaScript и HTML.

Библиотека jQuery помогает легко

получать доступ к любому

элементу DOM, обращаться к

атрибутам и содержимому

элементов DOM, манипулировать

ими. Также библиотека jQuery

предоставляет удобный API для

работы с AJAX. Сейчас разработка

jQuery ведётся командой jQuery во

главе с Джоном Резигом.
